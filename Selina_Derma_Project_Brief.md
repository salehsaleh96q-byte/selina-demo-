# مشروع: موقع عيادة Selina Derma

## نظرة عامة
موقع تعريفي لعيادة عناية بالبشرة (بوتكس، ميزوثيرابي، عناية بالبشرة). الهدف: هوية بصرية جذابة، احتراف طبي، وتحسين ظهور بمحركات البحث (SEO). الحجز يكون عبر زر واتساب/اتصال مباشر — بدون نظام حجز معقد.

**التقنية المطلوبة:** Next.js 14+ (App Router) + TypeScript + Tailwind CSS
**اللغة:** ثنائية (عربي RTL + انكليزي LTR) — مع مبدّل لغة بالهيدر
**الصفحات:** الرئيسية / الخدمات / عننا / تواصل

---

## 1. نظام الألوان (Design Tokens)

```css
:root {
  --color-white: #FFFFFF;
  --color-cream: #FDF8F6;       /* خلفية أساسية ناعمة */
  --color-rose-gold: #B76E79;   /* اللون الأساسي للهوية */
  --color-rose-gold-light: #E8C4C0; /* للخلفيات والتفاصيل الخفيفة */
  --color-rose-gold-dark: #9A5560;  /* للنصوص/الأزرار عند hover */
  --color-charcoal: #2B2520;    /* النص الأساسي (ليس أسود بحت) */
  --color-charcoal-soft: #6B5F58; /* نص ثانوي */
}
```

استخدم هذي القيم كـ CSS variables أو ضيفها بـ `tailwind.config.ts` تحت `theme.extend.colors`.

## 2. التايبوغرافي

- **العربي (Display - العناوين):** `Markazi Text` أو `Aref Ruqaa` (وزن خفيف للعناوين الكبيرة فقط)
- **العربي (Body - النص):** `Tajawal` (أوزان 300/400/500)
- **الانكليزي (Display):** `Cormorant Garamond` (للعناوين، يعطي طابع طبي-أنيق وليس تجاري صارخ)
- **الانكليزي (Body):** `Inter` (أوزان 300/400/500)

استورد الخطوط من Google Fonts بـ `next/font/google`. تأكد العناوين الكبيرة تستخدم `font-weight: 300-400` فقط (خفيف وأنيق، ليس bold ثقيل) — هذا جزء من هوية "العناية الراقية".

## 3. الستايل العام
- Minimal، مساحات بيضاء واسعة (whitespace generous)
- أيقونات line-art بسيطة (لا أيقونات مملوءة/ثقيلة) — استخدم مكتبة `lucide-react`
- لا حواف حادة صارمة ولا دوائر كاملة — استخدم `border-radius` متوسط (8-16px) يعطي نعومة بدون مبالغة
- لا أرقام تسلسلية (01/02/03) على الخدمات — لأنها ليست خطوات متسلسلة، هي قائمة خدمات متوازية

## 4. العنصر التوقيعي (Signature Element)
خط رفيع (hairline, 1-2px) بتدرج Rose Gold يظهر بحركة "رسم" (draw-in animation) تحت العنوان الرئيسي عند تحميل الصفحة الرئيسية — يرمز لدقة وحرفية العيادة. استخدمه بشكل خفيف ومحدود (الصفحة الرئيسية فقط، تحت الـ H1)، لا تكرره بكل صفحة حتى يحافظ على قيمته كعنصر مميز.

```tsx
// مثال تنفيذ مبسط بـ CSS
.signature-line {
  width: 0;
  height: 2px;
  background: linear-gradient(90deg, var(--color-rose-gold), var(--color-rose-gold-light));
  animation: drawLine 1.2s ease-out forwards;
  animation-delay: 0.3s;
}
@keyframes drawLine {
  to { width: 120px; }
}
```

## 5. هيكل الصفحات

### أ) الصفحة الرئيسية (Home)
1. **Header**: شعار Selina Derma (يسار بالـ LTR / يمين بالـ RTL) + روابط (الرئيسية، الخدمات، عننا، تواصل) + مبدّل لغة (AR/EN) + زر CTA "واتساب" بارز بألوان Rose Gold
2. **Hero Section**: عنوان كبير (مثال AR: "جمالك يبدأ من بشرتك" / EN: "Your Skin, Your Confidence") + خط توقيعي تحته + جملة فرعية قصيرة + زرين: "احجز موعدك" (واتساب) و "تعرف على خدماتنا" (يذهب لصفحة الخدمات)
3. **شريط ثقة (Trust bar)**: 3-4 نقاط مختصرة بأيقونات (مثال: "أطباء معتمدون"، "تقنيات حديثة"، "خصوصية تامة")
4. **معاينة الخدمات**: 4-6 بطاقات (بوتكس، ميزوثيرابي، عناية بالبشرة، تقشير، فيلر... إلخ) بأيقونة + عنوان + وصف سطرين + رابط "التفاصيل" يوصل لصفحة الخدمات
5. **قسم "لماذا Selina Derma"**: 3 أعمدة (خبرة، تقنيات حديثة، تجربة شخصية مريحة)
6. **CTA ختامي**: خلفية Rose Gold فاتح + دعوة للتواصل + زر واتساب كبير
7. **Footer**: شعار صغير + روابط سريعة + معلومات تواصل (واتساب، رقم هاتف، إيميل، الموقع على الخارطة) + أيقونات سوشال ميديا

### ب) صفحة الخدمات (Services)
- عنوان رئيسي + جملة تعريفية قصيرة
- شبكة كاملة لكل الخدمات (بوتكس، ميزوثيرابي، تنظيف بشرة، تقشير كيميائي، فيلر، علاج حب الشباب... عدّلها حسب خدمات العيادة الفعلية)
- كل خدمة: أيقونة + اسم + وصف فقرة (3-4 أسطر) + قائمة فوائد مختصرة (bullet points) + زر "استفسر عن هذي الخدمة" (واتساب مع رسالة معبأة مسبقاً تذكر اسم الخدمة)
- قسم أسئلة شائعة (FAQ) بأسلوب accordion بسيط

### ج) صفحة عننا (About)
- قصة العيادة (فقرة) — اكتب محتوى مبدئي مناسب، صالح يقدر يعدله بعدين
- نبذة عن الطبيب/الفريق الطبي (صورة + اسم + تخصص + سنوات خبرة) — استخدم placeholder حالياً
- قيم العيادة (3 نقاط: السلامة، النتائج الطبيعية، الخصوصية)
- صور من العيادة (gallery بسيطة) — placeholders

### د) صفحة تواصل (Contact)
- معلومات التواصل بشكل واضح وكبير: واتساب (بأيقونة)، اتصال مباشر، إيميل
- خريطة Google Maps مدمجة (iframe embed) لموقع العيادة بـ Fayetteville
- ساعات العمل (جدول بسيط)
- نموذج تواصل اختياري بسيط (اسم، رقم، رسالة) — يرسل عبر mailto أو يربط لاحقاً بخدمة مثل Formspree

## 6. SEO — متطلبات أساسية لازم تنفذ

```tsx
// في كل صفحة (app/[page]/page.tsx) ضيف metadata:
export const metadata: Metadata = {
  title: "Selina Derma | عيادة العناية بالبشرة والتجميل في فايتفيل",
  description: "عيادة Selina Derma — بوتكس، ميزوثيرابي، وعناية متخصصة بالبشرة في فايتفيل، نورث كارولاينا. احجزي موعدك الآن.",
  keywords: ["بوتكس فايتفيل", "عناية بالبشرة", "ميزوثيرابي", "Botox Fayetteville", "skincare clinic NC"],
  openGraph: {
    title: "Selina Derma",
    description: "...",
    images: ["/images/og-image.jpg"],
    locale: "ar_AR",
  },
};
```

نقاط إلزامية:
- `<html lang="ar" dir="rtl">` بالنسخة العربية (وعكسها بالانكليزية لو سويت subpaths منفصلة `/en`)
- استخدم tag واحد `<h1>` فقط بكل صفحة
- كل صورة لازم لها `alt` وصفي (مثال: `alt="عيادة Selina Derma لحقن البوتكس في فايتفيل"`)
- ملف `sitemap.xml` و `robots.txt` (Next.js يدعمهم تلقائياً بـ `app/sitemap.ts` و `app/robots.ts`)
- Schema markup نوع `LocalBusiness` / `MedicalClinic` بـ JSON-LD بالـ layout الرئيسي — يساعد كثير بظهور Google Maps والبحث المحلي
- صور بصيغة `.webp` ومضغوطة، واستخدم `next/image` دوماً (lazy loading تلقائي)
- سرعة تحميل: لا مكتبات ثقيلة غير ضرورية

## 7. الشعار
الشعار موجود جاهز عند صالح. حط الملف بـ:
```
public/images/logo.png   (أو logo.svg إذا متوفر — أفضل للجودة)
```
يفضل يكون بخلفية شفافة (transparent PNG) حتى يندمج مع الهيدر الأبيض/الكريمي بدون مربع حوله.

استخدمه بالهيدر والفوتر:
```tsx
import Image from "next/image";
<Image src="/images/logo.png" alt="Selina Derma" width={140} height={48} priority />
```

## 8. زر واتساب (العنصر الأهم تجارياً)
زر ثابت (floating) بأسفل الشاشة على كل الصفحات + أزرار CTA بمواقع متعددة. الرابط:
```
https://wa.me/<رقم_العيادة>?text=مرحباً، أريد الاستفسار عن خدماتكم
```
(صالح لازم يعطي الرقم الفعلي للعيادة قبل النشر النهائي)

---

## 9. الاستجابة (Responsive)
كل التصميم لازم يعمل بسلاسة على الموبايل (أغلب الزوار رايح يجوا من الموبايل عبر إنستغرام/واتساب). تأكد:
- الهيدر يتحول لـ hamburger menu بالموبايل
- زر الواتساب العائم ما يتعارض مع محتوى الصفحة
- البطاقات تتحول لعمود واحد بالشاشات الصغيرة
